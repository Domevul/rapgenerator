# ラップバトルゲーム

## 概要

このプロジェクトは、Phaser.jsとTypeScriptを使用して開発されるブラウザベースのリズムアクションゲームです。
プレイヤーは、事前に選んだ歌詞をビートに合わせて発動させ、対戦相手とのラップバトルを繰り広げます。
